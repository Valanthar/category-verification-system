# 🏅 Sistema de Verificação de Categoria / Category Verification System

| 🇧🇷 Português | 🇺🇸 English |
| :--- | :--- |
| **Descrição** | **Description** |
| Este é um aplicativo de console desenvolvido em **C#** que verifica a categoria esportiva de um usuário com base em sua idade. | This is a **C#** console application that verifies a user's sports category based on their age. |
| **Funcionalidades** | **Features** |
| - Validação de entrada (impede letras e valores inválidos).<br>- Classificação em categorias: Infantil, Juvenil, Profissional e Sênior.<br>- Validação da resposta S/N com feedback para entradas inválidas.<br>- Interface colorida para melhor experiência.<br>- Permite múltiplas consultas em sequência. | - Input validation (prevents letters and invalid values).<br>- Classification into categories: Children, Youth, Professional and Senior.<br>- Y/N answer validation with feedback for invalid input.<br>- Colored interface for better user experience.<br>- Allows multiple queries in sequence. |
| **Tecnologias** | **Technologies** |
| C#, .NET SDK, Git/GitHub. | C#, .NET SDK, Git/GitHub. |

## 📋 Lógica de Programação / Programming Logic

| 🇧🇷 PT-BR | 🇺🇸 EN-US |
| :--- | :--- |
| O sistema utiliza uma cadeia de `if/else if/else` otimizada para classificar a idade do usuário, eliminando condições redundantes. Um loop `while` garante que entradas inválidas sejam rejeitadas tanto na idade quanto na resposta S/N. | The system uses an optimized `if/else if/else` chain to classify the user's age, eliminating redundant conditions. A `while` loop ensures invalid inputs are rejected both for age and the Y/N answer. |
